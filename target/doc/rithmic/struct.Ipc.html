Expand description

Inter-process communication via pipes with a spawned child command

The child’s stdin & stdout can be accessed via the tx and rx fields. See also the input() and input_line() convenience methods. The child process is killed (if still alive) when this object is dropped

Examples

use std::io::Write;
use std::process::Command;
use proconio::input;

let mut ipc = Ipc::new(
    Command::new("python3")
        .arg("-c")
        .arg("print(int(input()) + 11)")
);

writeln!(&mut ipc.tx, "23");
input! { from ipc.input_line(), x: i32 };

assert_eq!(x, 34);
