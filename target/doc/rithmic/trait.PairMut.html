Expand description

Mutably borrow two indexes of a slice simultaneously.

Examples

ⓘ
let mut v = vec![vec![1, 2], vec![3, 4]];

// borrow conflict
v[0].append(v[1]);
let mut v = vec![vec![1, 2], vec![3, 4]];

let (x, y) = v.pair_mut(0, 1);
x.append(y);

assert_eq!(v, vec![vec![1, 2, 3, 4], vec![]]);

Safety

slice::split_at_mut is used to enable this method without unsafe
