Expand description

Mutably borrow two indexes of a slice simultaneously.

Examples

This does not work:

ⓘ
let mut a = [vec![1, 2], vec![3, 4]];

// borrow conflict
a[0].append(&mut a[1]);

but this does:

use rithmic::PairMut;

let mut a = [vec![1, 2], vec![3, 4]];

let (x, y) = a.pair_mut(0, 1);
x.append(y);

assert_eq!(a, [vec![1, 2, 3, 4], vec![]]);

Safety

slice::split_at_mut is used to enable this method without unsafe
